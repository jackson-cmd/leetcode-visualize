## 如何使用

```bash
# 安装依赖
pip install Pillow

# 生成所有动图
python3 generate_all.py

# 或生成单题
python3 problems/001_合并交替字符串/solution.py
```

└── problems/

    │   ├── README.md      # 题解
    │   ├── solution.py    # 可视化脚本

    │   └── ...

        └── ...
